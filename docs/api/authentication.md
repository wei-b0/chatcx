# 🔑 Authentication

All API requests must include an API key.

## 🔹 Header Example:
```http
x-api-key: YOUR_API_KEY
```

🔐 **Types of API Keys**
- **Admin Keys** → Full access (API Key management, querying).
- **App Keys** → Limited access (Only querying).
